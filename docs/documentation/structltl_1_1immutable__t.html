Detailed Description

template<typename T, typename Friend = void>
struct ltl::immutable_t< T, Friend >

The immutable_t struct.

This structure is used when you want your structure like class have some const attributes. Indeed, let's say you want to have something near like

struct Person {
const std::string name;
};

The name attribute is clearly const, however, if you want to move a Person, you will have a performance impact since the name will not be moved but copied

If you declared the name without the const qualifier, the name will be mutable, and it is not a good thing too.

The solution is to do

struct Person {
ltl::immutable_t<std::string> name;
};

Now, name is movable, but immutable for other cases. You will not be able to modify it, just move it.

If you want to have the possibility to modify it from your class, you may use the template argument Friend and the mut ref function

struct Person {
void make_name_bold() {
name.mut_ref() = "**" + *name + "**";
// or
name = "**" + *name + "**";
}
ltl::immutable_t<std::string, Person> name;
};
