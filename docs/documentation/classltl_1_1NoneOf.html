template<typename... Ts>
class ltl::NoneOf< Ts >

The NoneOf condition helper.

It is a simple helper to simplify condition You can use it like this:

int a = 18;
auto [b, c, d, e, f] = getSomeValues();
if(a == ltl::NoneOf{b, c, d, e, f})
// Here a is equal to none of b, c, d, e or f.
