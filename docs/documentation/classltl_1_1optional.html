Detailed Description

template<typename T>
class ltl::optional< T >

The optional class.

It is the same as std::optional<T> However, it provides a map function, and and_then function. The map function is the functor operator The and_then is the monad operator

int square(int);
ltl::optional<int> f(int);
ltl::optional<int> x;
auto y = x.map(square); // y = square(*x) or nullopt if x is empty
auto z = x.and_then(f); // z = f(*x) or nullopt if x is empty

It provides also a better value_or function in the sense that it has a default parameter type. It leads to the possibility to write

ltl::optional<int> x;
x.value_or({});

If coroutines are enabled, it also provides the co_await monadic operator

ltl::optional<int> f();
ltl::optional<int> g() {
co_return co_await f() * 2; // return f() * 2 or nullopt if f() returns a nullopt
}
